Resource Disambiguator Web (RDW)
================================

Prerequisites
-------------
 * Java 1.6+
 * Grails 2.3.8 
 * Postgres 8+
 * resource_disambiguator (https://github.com/SciCrunch/resource_disambiguator.git)


Getting the code
----------------

    cd $HOME
    git clone https://github.com/SciCrunch/rdw.git
    cd $HOME/rdw

Building
--------


    cd $HOME/rdw
    grails clean
    grails war
